About Quickie

Quickie is a macOS menu bar app for getting reminders into Apple's Reminders app before the thought disappears. Click the status item, fill in the reminder form, and save straight into a selected Reminders list without opening the full Reminders UI first.

Quickie is built around a compact reminder form, a status bar presence, a right-click utility menu, help content, and a configurable global shortcut so it can be summoned from anywhere on the desktop.

Version Quickie 1.0 (1). This help content is aligned with that app version.

Quick Start

  1. Click the Quickie icon in the menu bar.
  2. Use the global shortcut, ⌃⌥⌘Space by default, to open Quickie from anywhere.
  3. Enter a title, due date, due time, urgency, Reminders list, and tags.
  4. Press Return or choose Add to create the reminder.
  5. Choose Cancel or press Escape to clear the form without adding anything.

Fields

Title starts as New Quickie. Replace it with the reminder text you want to see in Reminders.

Date starts on today's local date. Time starts at the next whole hour.

Urgent uses a checkbox. Turning it on creates a high-priority reminder.

Organisation / List chooses the Reminders list where Quickie will save the item.

Tags starts with Quickie. Separate multiple tags with commas or spaces. Quickie saves them as hashtags in the reminder notes.

Global Shortcut is available in Settings from the Quickie menu bar menu. Use the shortcut recorder to press the key combination you want.

Settings

Enable shortcut turns the global hot key on or off.

Shortcut lets you record a custom key combination. Use Reset to Default to return to ⌃⌥⌘Space.

Reminder Defaults controls the values Quickie prefills when you open the form, including title, date, time behavior, urgency, default Reminders list, and tags.

Reset Reminder Defaults restores the default Quickie values: New Quickie, today's date, the next whole hour, urgency off, the Reminders list, and the Quickie tag.

Menu

Right-click the Quickie menu bar icon to open the utility menu.

About Quickie opens the standard About dialog.

Settings opens the Quickie Settings window for shortcuts and default reminder values.

Help opens this help page, including Quick Start and troubleshooting guidance.

Quit Quickie closes the app and removes it from the menu bar until you launch it again.

Troubleshooting

If lists do not appear, open System Settings, choose Privacy & Security, then Reminders, and allow Quickie full access.

After granting access, reopen Quickie or use the in-app Retry button beside the Organisation / List field.

If Add is disabled, make sure a writable Reminders list is selected.

If tags do not appear in the native Reminders Tags field, check the reminder notes. Apple's public Reminders API does not expose the native Tags field.

If a reminder does not appear immediately, open Reminders and check that the selected list is visible and syncing.

If the global shortcut warning appears in Settings, choose a different key combination. Another app or macOS may already be using it.
